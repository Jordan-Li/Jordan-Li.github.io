前言

在王者荣耀打了两个多赛季的辅助了，期间不断在B站和NGA搜索各种教程和攻略学习，并且经常观看虎牙全能辅王鑫鑫直播，再根据自己的一些实战的经验，总结了一篇辅助的综合攻略。虽然自己也不是完全精通了这个位置，但一些实战经验还是希望能够记录下来，故作此文。

辅助对于全局的影响力

抢夺资源，打破平衡

王者中最重要的两个资源：塔，经济。那个队伍能快速推塔，那个队伍能掠夺更多经济，就会有高几率获得游戏胜利。每个队伍上来的塔，经济都是一样。如何能打破这个平衡？下面有三个例子：（1）通过个人英雄技巧单杀对方。（2）多人压塔，借助兵线人数血量优势去推塔。（3）和队友入侵地方野区反野，反一个野怪等于取得双倍野怪的经济优势。 第一点是线上英雄需要做的事情，而二三点就需要辅助去配合队友完成。

占据视野，抢占节奏

辅助作为一个节奏位，要做的首要事情就是提供视野，抢占节奏。辅助主要联动的对象就是中野（除了明世隐这类挂件英雄可以跟随射手打出发育路优势）。对于中单而言，可以利用视野抢占线权，使得中单的支援速度得到提升。例如：中单去边路支援后，看到中线快到了，于是想回去清线，心想“可是对方中辅没来这边支援，也不在线上露头，我好慌怎么办”。于是绕路，漏兵，线权还抢不到，对面中辅发现你绕路立马抢线权立马支援别的边路去打优势了。对于打野而言，可以利用视野确定GANK的路线，帮助边路快速取得优势。例如：在开局确定对面打野的BUFF开，从而确定对方打野的GANK的路线，这样我方打野就可以根据对面的GANK路线来调整我方的GANK路线，使得GANK成功率得到提高。

保证前期，以多打少

以多打少的前提是快速支援。一般指中野辅联动在中路清兵之后，下一波兵来之前搞事情。双边快速清线后也可自己支援中路，或者反小野。中路英雄在清线后上下支援，入侵野区都很方便，这是前期的节奏点。对辅助来说，游戏前期重要的几件事：确定中路河道一侧的草丛安全与否，帮助中路快速清线，观察对方打野红开蓝开，保护己方打野到4级，然后和己方中野联动。比如在中路选择上，我方的清线速度不如对面，就需要团队配合来弥补，辅助就要发挥作用。

总结一句话就是：没有辅助，视野一片黑暗，中单清线塔下挂机，打野不敢gank导致万年野，边路等待对面gank等死。

辅助通用思路&攻略

关于铭文的选择

对于大多数辅助来说，万金油的铭文是“千血”铭文：宿命调和虚空。对于少部分需要经常利用技能的辅助，可以考虑带一小部分的冷缩。

在载入界面需要思考的内容

  1. 对面的控制强度，考虑要不要出抵抗鞋，一般来说抵抗鞋是必出的，但如果对面控制只有一个/两个软控时可以考虑先不出。
  2. 对面输出是否比较依赖平A，要不要补护甲鞋。如果前两点都有的话，可以出两双鞋子，因为鞋子对于其他装备的性价比高出很多。
  3. 对面的魔法伤害比例是否很高，需不需要把魔女的优先级提高。有一些比较容易忽略的魔法伤害：肉边的红莲斗篷、亚瑟的大招、射手的电刀伤害、狄仁杰老虎伽罗公孙离的魔法伤害、雅典娜的技能伤害、法辅的伤害等等。
  4. 第一件大物抗装出不详还是冰心，基本上有吕布，后羿，马克时优先不详。
  5. 光环装的选择：己方英雄输出是否依赖平A，多就出极影，少可出近卫。

辅助装的选择，升三的时机

目前版本光环装用处极大，混助攻经济、保KD，打团光环多人受益，既是为自己买的装备，也是为队友买的装备。这个版本辅助推荐双辅助装，光环+主动技能的搭配能为队伍增强不少功能。

  1. 近卫：卖点，后期可以提供144法抗，让对面中单的88+75法穿不至于趴光脆皮队友的法抗，而且这个版本法抗装备选择太少了，法抗的优先级大大提升。
  2. 极影：极影的固定属性前期对很多英雄都很重要，不但增加攻速还能减少冷却。
  3. 绿盾：第二辅助装的最好选择，如果第一件辅助装是光环，那么第二件辅助装基本都是绿盾。
  4. 奔狼：强开或机动性辅助的第二件辅助装的选择。例如：东皇，孙膑，鬼谷，太乙。

升三时机：一般不裸三，肉辅一般在出了两~三个小件之后升三，不推荐太晚升三了；法辅（除蔡文姬比较依赖经济）不太依赖装备，所以三级辅助装可以晚点升，可以在15级满级了再升。还有一个判断点是一般队友经济达到8000-9000的时候就可以考虑将辅助装做大。或者自己急需某件装备（例如魔女）的时候，可以考虑将辅助装做大。

正确的抗伤的姿势

逆风局，总体来说是真的很难扛住，但是再难扛，也总有个正确的姿势，去如何尽量少扛，如何让自己扛得有价值，没有价值的扛伤就是送。

正确的扛伤姿势就是：尽量让你在扛伤害的时候，你队友也在输出，这就是所谓的创造输出环境。不该吃的伤害、能躲的伤害控制尽量躲，所以并不是你一个位移跳进对面阵容里就是在扛伤，我会告诉你，辅助的位移基本都是用来躲伤害的，闪现逃命比闪现开团用得多！

所以正确的站位应该是，打团尽量不要离开C位一半射程的距离，这样保证自己的技能可以保到C位，自己C位也不觉得脸上没肉而不敢上去输出。很多人玩肉就是经常会抱怨：我上了，队友怎么不上啊？这时候你需要思考一下，是不是你上的那个位置，和队友已经脱节了，你一个位移跳到了对面C的脸上，然后不仅长手的C，连手短的战士辅助都能输出到你；而队友射手需要多走几步上前才能上去A，而且等他上到去的时候，你已经残血了，射手已经觉得没有什么输出空间了，于是又后撤了；又或者，由于你的太深入，导致C位也跟着深入进入到了对方中单的技能范围内，对方战士刺客的突进范围内。

最后还有正确的出装，不要跟着系统推荐的装备出，要思考，这个英雄有什么特点，出什么肉装可以让他更能扛伤害。比如：夏侯、刘邦、张飞的盾是血量最大百分比，那么我们是不是优先考虑带血量的防御装。

如何正确的开视野

很多人以为逛河道逛野区多游走逛草丛就是开视野，但是其实很多时候做的是无用的视野，以及误导队友走位的有害视野。有害视野的问题比较严重：路过草丛不去探，觉得没人打你应该没人，然后带着队友一直深入，结果后方传来噩耗。这通常发生在中路附近河道之间的草丛之中，因为那里好多个草丛，你不去全探，你是不能保证对面会不会5个人全蹲在那个小草那里，不要以为你占了中路大草发现没人，就以为没人了，很多时候尤其是高端局，河道的小草是最凶险的蹲位。另一种有害视野是中后期没有防御塔的情况下自己探得太过深入，导致队友远远脱节，这时候不是你死，就是你队友因为附近没有你占视野而死。所以正确的开视野的姿势：

  1. 不要和队友过分脱节，通常队友在你身后一两个草丛的距离是正确的；前期去对方红区看对面野辅导致有点脱节也无所谓，因为前期大家都有塔，都很忙，没空管你。
  2. 路过能探的安全草丛尽量去探，感觉不安全的放技能去探，或者在你有逃命技能的情况下，你去那个草附近来回走一下看看有没有人打你，两秒中不打你证明这个草顶多只有一个人蹲，这时你可以进去和他抢视野；如果有三四个人蹲，基本上一看到你就POKE你了，你也没机会进去抢视野。
  3. 逆风局探红野区需谨慎，随时留技能逃命。另外一点就是在逆风局的时候，如果已经明显经济落后的情况下，就需要避战。这个时候已经不需要在去进攻草探视野，只需要在防守草占住视野，保证队友在对面无法直接gank到的地方平稳发育即可。
  4. 顺风局或者均势局对面都不露头或者预料到对方会往这里来的时候，请沉住气，占稳你所在的草丛，不要占了这个草发现没人就走了，很多时候你站稳了，你能配合后来赶上的队友蹲一波；或者你占了一个比较阴险很少人会探的草丛(敌方中塔右下角那个)，很可能会看到对方几个人的走动方向，然后你可以配合队友包抄。顺风局切记要站稳草丛！因为可能你一露头让对面看到，对面就怂得跑了，所以要沉住气站稳草丛！
  5. 关于抢视野：很多人不会反制鬼谷子，其实很多原因就是抢视野不够强势，体现在哪里呢？前期看着鬼谷占了一个草丛视野，你就不敢去抢了。但如果玩鬼谷碰到会玩的牛魔就很头疼，因为牛魔占视野的强度确实比鬼谷高，牛魔占住了我那边的河道草丛，我一旦去游走，他们全队都看在眼里，这时候我只能选择绕路。别问我为什么不去和牛魔1V1抢那个草，因为我抢不过他，我12技能交了对方中单过来我就没有逃命技能了，牛魔基本只会用一技能和你互动。肉辅都是抢视野的好手，对上鬼谷这种伪前排半肉半功能辅，可以把鬼谷子压得很惨。常见克制鬼谷的肉辅有：张飞、牛魔、廉颇、鱼。

应该打先手还是打后手

在低端局，队友通常都是很莽的那种，就是冲冲冲，这时候能先手到敌方C位的情况下我很推荐主动先手；在高分局，往往就是打拉扯，团战打个几十秒，或者是互相交技能摩擦之后都走了，所以高分局打后手抓失误是更加稳健的打法。

对于不同开局的处理

双方都不进行入侵的正常开局

首先注意对方可能一级蹲中一血的英雄：鬼谷，太乙，关羽，盾山，如果对面有，开局提醒中单。正常开局先从红区走到中路草丛，帮中单抢2级线权，然后去看对面的蓝BUFF还在不在，打字“对面打野X开抓X”，然后去看自家蓝有没有被偷，再帮中抢一波线。

然后再根据敌我打野的开局位置情况，作出举动思考 ：

  1. 如果我方是蓝开，对方是红开，这样大家刷完一波野最终可能会在我方下路相遇，这时候你可以帮中清完第二波线后就去帮打野加速到四，这样可以抢占先机抢到下路草丛。我只推荐是我家蓝开对方红开的情况下去帮打野到四，因为通常蓝开是比红开快(红开要绕路，不然就漏鸟慢点到四)，这样可以制造速度差反蹲对面一波打开局面。
  2. 如我方打野红开，对方打野蓝开，那么你需要在1分15秒就去到上路草丛，（一般没有辅助跟随的情况下打野1分30秒刷完全部野区）通常会看到对面野辅在打鸟，而你方打野还在打蓝，这时候你要做的是，发撤退，让他别来，你退回中到上路河道之间的草丛做视野。由于你这时候报出了对方打野的视野，那么会玩的上路就会苟，会玩的中野就会和你一起反蹲对面打野，而对方辅助做少了视野，就会导致打野地图一片黑暗，十分难受。
  3. 我方打野和对方打野都是同一个BUFF开，那么意味着4级大家不会相遇，这时候高端局基本上都是各抓边，双方一直压塔直到第二波野怪刷新。如果双方到4都不抓边，那么就会在中路一边占一个草丛，谁先露头谁先手谁就是弟弟（抛开阵容的话，中路这一波3v3基本是谁先手谁劣势）。

入侵/反入侵的开局

首先需要判断很可能打入侵的阵容：裴擒虎、刘备、云中君、兰陵王、盘古、橘右京、牛魔、太乙、鬼谷、东皇、扁鹊、夏侯、孙策等前四级作战能力非常强的组合；如果双方都有这几个英雄，通常都是不入侵或者互换BUFF的开局。

再来判断前期作战能力弱守不住野区的英雄：李白、猴子、荆轲、娜可露露、孙膑、虞姬……通常那种英雄哪边多哪边就会被入侵，好的情况下能换BUFF，不好的情况下丟BUFF丟人头。如果要入侵，判断入侵哪个BUFF比较好打：如对方选出的双边是孙策+虞姬，那么入侵的时候可以和队友说一声，前期侧重在虞姬的这边入侵，避开孙策，以增加入侵的容错率。

开局守野区具体做法：

上单在蓝BUFF草丛蹲到30秒发现没人来才上线，而你和打野(甚至中单)蹲在红上面草丛拿视野，开局没人来才打野。此时如果对方来反蓝，上单就怂回上塔，我方打野直接去对面换蓝，辅助和中单清中去打野那一边支援，另外对方蓝区最深入的那组狼怪基本不能要，不然很容易被包崩盘。由于前期被反的一方都是阵容前期弱势的，能换到对面蓝BUFF已经不错了，对面来包的话切勿再打小野。如果对方来反红，你们几个人蹲在红的那个草丛，对方集堆过来反红无防备的时候很可能被你们一起控中被你们先手一套(尽管你们先手后可能依然打不过，但是如果对面核心位置被打残了就会跑)；如果你们控制这一波控制交不好，就要第一时间跑，赌不到这一波对面就要3BUFF开了。

什么时候应该帮中路抢线权

一般情况下：前期直接露头上线和中单一起清线；中后期队友伤害高的情况下，你蹲一边草丛做视野蹭经济就好了。

什么时候坚决不和中单一起清线：当你从别的路支援回来、从泉水满状态出来的情况下，如果对面已经把中线推过来，然后就走了不压中，你中单单独清线的时候，你最好不要去和他一起清(除非第二波兵线快来了)。因为本来对面先把线推过来了，我方线权已经丢了，你再去，这不是在抢线权，而是单纯地蹭经济。这时候你暴露了视野，错过了第一时间去支援打野、帮边的时机，很亏。还有一种情况就是，我方中辅打不过对方中辅时候(尤其是前期开局)，就不要头铁去抢了，弄不好容易送头，这时候去帮打野收益比较大。

当你是太乙鬼谷时，抢中也可以是另外一种方式：当兵线交汇时，我方法师上线清，你在侧边蹲对面的中辅回归路线，（这个一般是在4级以后）对面走位失误的话，很容易造成击杀噢；如果对面比较谨慎你抓不到的话，才上线帮助清线。

帮助中路抢完线权后

首先说一个细节：清完第一波线后去看对面蓝BUFF,发现对面是红开，而这时候2级法师可能单独去了偷猪，这时候要提醒他撤退。

中辅联动：第一波兵线清完后的套路基本是很固定的，就是看对方打野开局，看自家另外一个BUFF有没有被偷。不建议第一波清完后不看视野就去跟中抓边和偷猪。每抢完一波中后，可以选择跟着中单一起抓边，也可以去带动中单反蹲对方支援的中辅，根据兵线的视野规则，清完兵后还有两秒左右的视野，这时候你们可以耿直地去一条边，给对方一个信息，从而导致他们上钩。

野辅联动：注意打野的位置，每当打野刷完一波野怪，或者一边野区的时候，这就是打野带节奏的时机了，这时候你可以去帮打野，因为有时候中单的节奏和打野的支援路线是不同的，而打野最先到四所以这时候跟打野收益最大。

刚才谈到了中野的节奏，但在我认为，中野辅都有自己的节奏，再清完线后，各有选择，不必强求谁跟谁的节奏，要自己去判断是自己去带节奏还是跟队友节奏，比如对方打野蓝开的例子，对面野区刷的速度比我们要快，对面蹲的时间基本上是比我方快的，这时候中野都不该去（除非确保打团能够打赢）如果你意识中知道这个事实，且你如果没来得及提前做好视野，你应该要做的就是提醒队友别去上路，而不是盲目地跟随他们的节奏。

什么时候可以单独游走去边路抓人

一般来说不推荐单独游走边路，鬼谷，太乙等抓边成本比较低的辅助另说。原因如下：

  1. 游走成本过高，很多情况就是，你去了，控制不到人。更重要的是你暴露了视野，中野失去了帮助，对面打团更加有底气。
  2. 一个被消耗半血的劣势边路，你一个辅助过去，顶多就是过去一起被压，清兵分经济，甚至还能送双杀。
  3. 优势边对面都很苟，除非你能稳定安全越塔杀，不然去那里蹲草收益不高，很多情况就是蹲到了对面中野辅过来，然后发一波撤退。

如果执意要单独游走边路，也要等到四级以后。这里重点提一下鲁班大师，这个英雄游走的成本虽然不低，但是抓边的效果很好，很容易开到对面抓死，帮助双边扩大优势。

正确认识和带动节奏

好的节奏就是在一波gank成功后，还能有兵线推塔。团战胜利后的优先级：推塔 > 反野 > 回城补血。一般的节奏就是边路gank成功，杀掉对方英雄，但是兵线却没有，对方野区也没有野怪。相反对方利用兵线人数优势推了另一侧的边塔。不好的带节奏就是先手开团失败给对方创造输出空间，导致先手开团英雄被快速杀掉，团战劣势甚至失败。

辅助的BP简谈

一二楼万金油：带控肉辅如：牛魔，张飞，廉颇，其他辅助在一二楼容易被对方针对或者被队友针对(比如一选孙膑小明，队友不选肉边，导致没人抗伤)。

  1. 什么时候选鬼谷子很好打：对面选了站撸射手、无位移法师的时候，必选鬼谷，如果对面再来个短腿战士打野，短腿战边，这些都是加分项。对面人均一个位移以上的时候不选鬼谷子，如公孙离，诸葛，李白，达摩之类。主要还是看对面双C有没有位移，双C都有位移，选鬼谷经常拉不到人。
  2. 什么时候选蔡文姬：对面缺少爆发英雄，对面有部分不带控有点伤害的英雄也可以：无控中单(诸葛，婉儿)、控制不强的战边，但对面有达摩、白起、鬼谷子时别选蔡文姬！
  3. 什么时候可以选庄周：首先鱼使用来counter对面群控的，如：甄姬王昭君钟无艳牛魔武则天白起……然后队友的中单最好带硬控，再不行打野或者上单要有控制，当队伍没有硬控的时候不推荐拿鱼。为什么：因为如果队友没有硬控的时候如果你辅助再不补充控制/开团的时候，这一局都开不到人。
  4. 什么时候可以选孙膑：队友有坦边的时候，同时对面野辅前期不是特别强的时候。队友是猴子李白荆轲等前4级特别弱的时候最好别选孙膑，同理对面打野是老虎刘备云中君的时候你也别选。孙膑最大的缺点就是前期弱。
  5. 对面拿到鲁班大师的时候：可以反手选一个廉颇。廉颇在释放技能的时候有霸体，不担心被鲁班大师开，同时具有群控，可以开到对面的C位，是一个很好反制鲁班大师的英雄。
  6. 阵容需要打前期入侵，选什么辅助：牛魔、太乙、东皇、鬼谷。
