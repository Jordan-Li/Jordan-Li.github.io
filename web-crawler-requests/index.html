网络请求（三）：Requests库的使用

Requests库——第三方库

虽然Python的标准库中urllib模块已经包含了平常我们使用的大多数功能，但是它的 API 使用起来让人感觉不太好，而Requests宣传是 “HTTP for Humans”，说明使用更简洁方便。Requests 是用Python语言编写，基于 urllib，但是它比 urllib 更加方便，可以节约我们大量的工作，完全满足 HTTP 测试需求。
