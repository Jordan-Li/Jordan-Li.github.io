如何让Windows桌面看起来更美观

 2020-02-18  2020-02-23
  Windows
  15 minutes read (About 2261 words) 0 visits

前言

虽然本人一向用的是Macbook，但是对于Mac的系统却一直不感冒，觉得十分麻烦。由于当时给Macbook分的区占掉了手提的一半，导致上半学期使用的时候Windows系统会出现内存不够用的问题。经过一番心理斗争，我将Mac系统直接删掉了，将内存全部转移到Windows这边来。但在使用了几个月以后，Windows的桌面的程序/文件越来越多，我开始寻找整理收纳的方法。

腾讯桌面整理

我最开始使用的是腾讯桌面整理对桌面的文件和文件夹进行分类归纳，归纳的效果图如下：

  • 使用这款软件的优点：
  1. 你可以在桌面直接点击你所有想要找到的文件/文件夹。（一步到位）
  2. 右键“整理”可以自动帮你把桌面上没有被收纳的文件自动收纳到桌面的格子中。
  3. 格子的大小/透明度/位置等等都可以自主调整，个性化程度也比较大。
  4. 双击鼠标左键可以一键隐藏桌面所有的文件，给你一个干净的屏幕。
  • 使用这款软件的缺点：
  1. 随着时间累积，每个格子中的文件会越来越多，显得冗杂。
  2. 随着格子数量增多，桌面有一大半的位置被占据，和没有做收纳之前的差别不大，十分不美观。
  3. 由于许多文件挤在同一个格子里面，要么扩大格子的大小，随之而来的就是缺点2，要么只能上下拉动寻找目标文件，搜索文件开始变得麻烦

总而言之：使用这款软件的初期体验是十分不错的，但随着时间的累积，体验感会逐渐下降，桌面被文件占据的空间依旧十分大。因此，在使用了三四个月该款软件以后，我决定寻找其他整理桌面的方法。随之我想到：虽然Mac的操作系统我十分不习惯，但是Mac的桌面我很喜欢，所以我决定搭建一个类似于Mac桌面的Windows桌面。

雨滴桌面

安装雨滴桌面

在对比百度/知乎/酷安等多处关于优化Windows桌面的文章以后，最后锁定了这款雨滴桌面。

雨滴桌面官网连接：雨滴桌面中文版下载

下载完以后安装即可，此时桌面就会有所变化，但并不是我们要的效果。
当然只有这个软件不足以让他如此优秀，还有社区朋友们各种神奇的皮肤和插件。因为本人想要搭建的是类似于Mac桌面的Windows桌面，所以采用了下面这款比较简洁的插件。

插件下载连接：曲水网盘(提取码：5bd3)

解压完成后将会得到4个文件。

前面两个文件直接双击打开安装即可。最后的雨滴时钟插件则需要解压缩后剪切文件夹到皮肤目录中。皮肤目录的打开方式可以在状态栏中右键雨滴桌面>皮肤>打开皮肤目录。

全部导入完成后，在状态栏中右键雨滴桌面>管理进入管理的界面。点击左下角刷新全部就可以刷新桌面。

配置桌面

把插件安装/卸载并且拖到自己想要的位置以后，就可以把允许拖动（在雨滴管理界面的右下角）给点掉，对于桌面时钟这类插件可以把点击穿透勾选上。

配置Dock栏

Mac系统桌面最大的特色就是除了Dock栏以外的地方是没有程序的，文件/文件夹放置在桌面。但有一点不足的是：如果想要桌面变得完全整洁，在Finder中寻找文件就变得十分麻烦。但利用雨滴桌面，我们可以把Mac系统桌面这一缺陷弥补，而配置Dock栏也成了美化桌面的重中之重。

配置图标

右键Dock栏可以看到列表文件这一选项，打开以后可以看到：

对于图标文件的修改，可以右键Dock栏点击更换图标，然后把想要更换的图标拖动到对应的程序上面即可。图标我用的是【Pure 轻雨 图标包】，在插件链接下载下来的文件中可以找到。里面提供了多达2700多个图标，一定能挑选到自己喜欢/匹配的图标替换掉。

配置地址

配置文件地址的时候麻烦事情就有点多了。主要有三种类型需要我们配置：

配置一般程序

对于普通的程序，我们可以直接找到程序的快捷方式，右键属性>快捷方式>目标,把目标栏中的地址复制出来，粘贴到Gamedir=后面即可，（记得把双引号去掉）。或者直接找到程序的源地址，将其输入进去即可，后缀一般为exe。

配置文件夹

对于文件夹，我们需要找到文件夹的地址，然后将文件夹的地址粘贴到Gamedir=后面即可，例如 D:\新建文件夹,这种方式的地址一般不带有后缀。点开Dock栏图标后可以直接打开文件夹。可以将一些不常用的程序/文件夹的快捷方式放置到同一个文件夹中配合使用。例如：

然后再在配置文件中某个图标对应的地址中输入：D:\不常用应用即可。

配置UWP程序

这里面最难搞的就是Microsoft商店里面下载下来的这些UWP程序，他们找不到明确的程序的地址，所以没有办法定位到源文件的位置，但是我们可以利用快捷方式巧妙解决这一个问题。我们可以对UWP文件在某个目录下创建快捷方式，然后将图标地址定位到这个快捷方式的地址。这种方式的地址以lnk结尾。例如：

我在D盘下创建了一个微信（UWP版）的快捷方式，随后我只要在配置文件中输入：D:\微信.lnk 即可将Dock栏中的图标导向UWP程序辣。另外对于回收站我们也可以用这种方式解决。

任务栏透明化

经过上面一番设置，我们已经可以将桌面美化地差不多了。但是唯独黑黑的任务栏十分碍眼。因此我们可以使用插件将Windows的任务栏透明化，用到的插件是StartsBack++。

StartsBack++下载：曲水网盘（提取码：ffgb）

直接下载安装然后根据喜好调整配置即可。最后我们右键任务栏，找到任务栏设置，将任务栏在屏幕上的位置设置到屏幕的上方，就可以得到一个高仿的Mac桌面啦！最后啰嗦一下，关掉桌面上面的回收站可以通过以下的方式：

右键任务栏>任务栏设置>主题>桌面图标设置>将回收站勾选掉

其余的一些优化Windows桌面的小插件可以自己探索，至少目前我已经得到了我理想的桌面~
